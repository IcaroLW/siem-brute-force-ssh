Este é um writeup que criei de um projeto de cibersegurança, ele foi feito em um homelab que possuo, duas máquinas foram usadas. Uma delas sendo Ubuntu Server com o Wazuh SIEM integrado e a outra uma máquina Ubuntu sendo um agente.

Objetivo: Um ataque de força bruta via SSH e validar a capacidade de detecção do SIEM Wazuh em tempo real, demonstrando o fluxo completo desde o ataque até a análise forense do log.
